dfemtoolz_multimaterial

Is the software used for merging multiple models (materials) into one multimaterial model.

In input/ directory are the input files, as well as config file for the program.

In output/ directory user can find the results. Output files can be different if user modify settings in config file.

Final results are zipped in output/
